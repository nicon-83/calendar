Функция-конструктор Calendar()

Данная функция написана на "нативном" javascript. При создании элемента необходимо в функцию передать объект, содержащий следующие параметры:

new Calendar ({
parentElem: родительский контейнер (объект, предварительно созданный на странице),
width: ширина контейнера календаря в px или % от ширины родительского блока,
numberBgImg: количество фоновых изображений, которые находятся в папке ./img/,
idImgWrap: id контейнера фоновых изображений,
idCalendarWrap: id контейнера календаря,
idPrevBtn: id кнопки назад,
idNextBtn: id кнопки вперед,
idLink: id ссылки для отображения текущей даты/года/диапазона лет,
idDayOfWeek: id таблицы содержащей дни недели,
idDateTable: id таблицы содержащей дни выбранного месяца,
idMonthTable: id таблицы месяцев,
idYearTable: id таблицы содержащей диапазон лет,
idYearRangeTable: id таблицы содержащей расширенный диапазон лет
});

Для добавления календаря на страницу нажмите кнопку "добавить календарь", все указанные выше параметры будут переданы в функцию автоматически. Количество календарей не ограничено. Добавленные календари можно удалить соответствующей кнопкой, которая появится после добавления первого календаря.
